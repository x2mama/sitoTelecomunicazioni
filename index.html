01 — Le Origini

Cos'è Arduino
& da dove viene

Arduino nasce nel 2005 all'Interaction Design Institute Ivrea, in Italia. Il progetto fu ideato da Massimo Banzi e un piccolo team di designer e ingegneri, con l'obiettivo di creare una piattaforma economica e accessibile per studenti, artisti e maker che volessero interagire con il mondo fisico attraverso l'elettronica.

Il nome "Arduino" viene da un bar di Ivrea frequentato dal team, che a sua volta era intitolato al re Arduino d'Ivrea del 1002 d.C. La scheda divenne rapidamente un fenomeno globale grazie alla sua natura open-source: sia l'hardware che il software sono liberamente disponibili, modificabili e redistribuibili.

2005

Prima scheda Arduino

Nasce all'IDII di Ivrea per rendere l'elettronica accessibile a non-ingegneri.

2008

Arduino Duemilanove

Prima versione ad alimentarsi via USB, diventa lo standard per i maker.

2010

Esplosione della community

Milioni di utenti in tutto il mondo, migliaia di librerie e progetti open source.

Oggi

Ecosistema globale

Dalle scuole alle startup, Arduino viene usato in robotica, IoT, arte interattiva e molto altro.

04 — Programmazione

Come si programma Arduino

Arduino si programma tramite l'Arduino IDE, un ambiente di sviluppo gratuito basato su un linguaggio derivato dal C++, semplificato per essere accessibile anche ai principianti. Il codice viene scritto sul computer e poi caricato sulla scheda via USB.

Ogni sketch (programma) Arduino ha sempre la stessa struttura di base: due funzioni obbligatorie che il sistema chiama automaticamente.

void setup() {
  // eseguita UNA sola volta
  // all'accensione o reset
}

setup()

È il "punto di partenza". Si usa per configurare i pin (INPUT o OUTPUT), inizializzare display, avviare la comunicazione seriale. Viene eseguita esattamente una volta.

void loop() {
  // si ripete all'infinito
  // finché c'è alimentazione
}

loop()

È il "cuore" del programma. Tutto ciò che scrivi qui viene ripetuto ciclicamente, decine di migliaia di volte al secondo, finché la scheda è accesa.

pinMode(8, OUTPUT);
digitalWrite(8, HIGH);
delay(1000);

Funzioni essenziali

pinMode configura un pin come ingresso o uscita. digitalWrite accende (HIGH) o spegne (LOW) un pin digitale. delay mette in pausa per N millisecondi.

05 — Il Codice

Il Semaforo:
codice completo

Il progetto collega tre LED (rosso, giallo, verde) ai pin digitali 8, 9 e 10 della scheda Arduino. Il programma li fa alternare simulando il comportamento di un vero semaforo stradale.

semaforo.ino Arduino C++ 
// Dichiarazione variabili: assegniamo un nome ai pin
int rosso  = 8;
int giallo = 9;
int verde  = 10;

// setup() → configurazione iniziale, eseguita una volta sola
void setup() {
  pinMode(rosso,  OUTPUT);  // pin 8 → uscita
  pinMode(giallo, OUTPUT);  // pin 9 → uscita
  pinMode(verde,  OUTPUT);  // pin 10 → uscita
}

// loop() → ciclo infinito, il cuore del semaforo
void loop() {

  // ── FASE VERDE ───────────────────────────────
  digitalWrite(verde,  HIGH);  // verde  → ACCESO
  digitalWrite(giallo, LOW);   // giallo → spento
  digitalWrite(rosso,  LOW);   // rosso  → spento
  delay(5000);                   // attendi 5 secondi

  // ── FASE GIALLA ──────────────────────────────
  digitalWrite(verde,  LOW);   // verde  → spento
  digitalWrite(giallo, HIGH);  // giallo → ACCESO
  digitalWrite(rosso,  LOW);   // rosso  → spento
  delay(2000);                   // attendi 2 secondi

  // ── FASE ROSSA ───────────────────────────────
  digitalWrite(verde,  LOW);   // verde  → spento
  digitalWrite(giallo, LOW);   // giallo → spento
  digitalWrite(rosso,  HIGH);  // rosso  → ACCESO
  delay(5000);                   // attendi 5 secondi

  // → il loop riparte dall'inizio: torna al VERDE
}
